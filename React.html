React Jobs Portal

Designed a React jobs website using React.js and Vite, utilised the React.js concepts for a responsive and real-time userinterface.

Implemented features for organizations to post React jobs, with options to filter by location and salary, along with an option to add new job postings.

All the Job postings are stored in the form of a JSON type on the server side.

The job posting can also be deleted if there are no openings in the company.

Technologies:
  • - React.js
  • - Vite
