Crop Prediction and Translation using Machine Learning and Artificial Intelligence

Developed a machine learning model for prediction of crops and trained the model using logistic regression algorithm with some part of the dataset fortraining and remaining fortesting

Ensembling Algorithms like XGBoost were used to increase the efficiency of the model so the accuracy of the prediction can be better.

A userinterface is developed using the streamlitwhere the user can enter the soil and environment values and predict the suggested crop

Generative AI gives some suggestions regarding the crop and language selection helps to select the suitable language for the details to be displayed.

Technologies:
  • - Python
  • - Streamlit
  • - Google Generative Ai
  • - Logistic Regression
  • - Google Translation
