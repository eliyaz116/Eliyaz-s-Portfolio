Student Information System

This is the first project i have done with my team, after learning the basic web development skills HTML, CSS, JavaScript during the academics.

Modules like Home page, About us, Contact us, login student page, Student Home Page, Login Page Admin, Attendance page, Marks page are included in it.

MySQL database is used for storing the students infomation and Php is used as backend in this project.
